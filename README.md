# Instrumental-Dataset
Dataset used to train a pure instrumental music AI source tracing model

AI Music Detection Dataset

License: CC BY-NC 4.0 (https://creativecommons.org/licenses/by-nc/4.0/)
GitHub Repository: https://github.com/evamooreshtg58-beep/Instrumental-Dataset

A large-scale, multi-source dataset for detecting AI-generated music and tracing generation models.
7,045 audio clips (30s each) | 4 sources (Human, MusicGen, Suno, Doubao) | 8+ musical styles

DATASET OVERVIEW

This dataset is designed to facilitate research on distinguishing human-composed 
music from AI-generated content, as well as identifying the specific generative 
model behind an audio clip.

Category    Source                   Clips    Sample Rate    Avg. Duration
----------------------------------------------------------------------------
Human       FMA Medium (instrumental) 1,417   44.1 kHz       30.0 s
AI          MusicGen (Meta)           1,412   32.0 kHz       30.0 s
AI          Suno                      2,808   48.0 kHz       30.0 s (trimmed)
AI          Doubao                    1,413   44.1 kHz       30.0 s
----------------------------------------------------------------------------
Total       -                         7,050   -              -

Note: The actual file count in the archive is 7,051, which includes metadata files.

All audio files are stored in WAV or MP3 format, and all have been trimmed/padded 
to exactly 30 seconds during feature extraction. The human subset covers diverse 
instrumental genres including blues, classical, jazz, electronic, and more.

REPOSITORY STRUCTURE

release_dataset/
    README.md
    file_list.csv                 (Metadata for all clips)
    human/                        (1,417 real instrumental tracks, by style)
        blues/
        classical/
        electronic/
        ...
    ai/
        musicgen/                 (1,412 clips from Meta's MusicGen)
        suno/                     (2,808 clips from Suno API)
        doubao/                   (1,413 clips from Doubao)
    AI_Music_Detection_Dataset.tar.gz   (Compressed archive, ~2 GB)

Metadata file (file_list.csv) columns:
    path    - Absolute or relative file path
    is_ai   - Binary label (0 = human, 1 = AI)
    source  - Categorical label (Human, MusicGen, Suno, Doubao)
    style   - Musical genre (e.g., blues, classical)

DOWNLOAD INSTRUCTIONS

The full audio dataset is available as a compressed tar.gz archive tracked with 
Git LFS in this repository. To download:

1. Install Git LFS (https://git-lfs.com/).
2. Clone the repository:
       git lfs install
       git clone https://github.com/evamooreshtg58-beep/Instrumental-Dataset.git
3. Extract the archive:
       tar -xzvf release_dataset/AI_Music_Detection_Dataset.tar.gz

If you only need the metadata, you can download file_list.csv directly from the 
repository without cloning the entire dataset.

INTENDED USE CASES

- Binary Classification: Human vs. AI-generated music.
- Multi-class Source Attribution: Identify which AI model (MusicGen, Suno, or 
  Doubao) produced a given clip.
- Feature Analysis: Investigate rhythmic regularity, spectral artifacts, or 
  deepfake fingerprints.
- Benchmarking: Evaluate robustness of detection models across different 
  generative architectures.

================================================================================
数据集概览
================================================================================

本数据集旨在支持区分人类创作与 AI 生成音乐的研究，以及识别音频片段具体由哪个生成模型产生。

类别      来源                    片段数量   采样率      平均时长
---------------------------------------------------------------------------
人类      FMA Medium（纯器乐）      1,417    44.1 kHz    30.0 秒
AI       MusicGen (Meta)          1,412    32.0 kHz    30.0 秒
AI       Suno                     2,808    48.0 kHz    30.0 秒（截取）
AI       豆包                      1,413    44.1 kHz    30.0 秒
---------------------------------------------------------------------------
总计      -                        7,050    -           -

注：压缩包内实际文件数为 7,051，包含元数据文件。

所有音频均以 WAV 或 MP3 格式存储，并在特征提取阶段统一截取或填充至 30 秒。
人类音乐子集覆盖了布鲁斯、古典、爵士、电子等多种器乐风格。

================================================================================
适用研究场景
================================================================================

- 二分类检测：区分人类音乐与 AI 生成音乐。
- 多分类溯源：识别音频片段出自哪个具体 AI 模型（MusicGen、Suno 或豆包）。

- 基准测试：评估检测模型在不同生成架构上的鲁棒性。
